## 📖 简介

`M-AIDE-Plus-Install` 是基于 **Aide Plus 官方安装器** 二次开发的 NDK 安装增强工具。

官方安装器在使用过程中存在 CMake 缓存异常、进度条体验差等问题，本项目针对这些痛点进行了优化，并额外加入了本地安装与内置 URL 下载功能，让 NDK 的安装更加简单、稳定。

---

## 🚀 快速开始

### 前置要求

- 已安装 **Aide Plus**
- Android 设备或模拟器

### 使用方式

1. 前往 [Releases](https://github.com/mizaawa/M-AIDE-Plus-Install/releases) 下载最新版本
2. 安装到设备
3. 选择安装方式：
   - **在线安装**：选择目标 NDK 版本，通过内置 URL 一键下载安装
   - **本地安装**：选择已下载的 NDK 压缩包进行本地安装

---

## 🔧 解决的问题

- ✅ 修复官方安装器 **CMake 缓存异常**导致安装失败的问题
- ✅ 优化文件操作时的**进度条显示**逻辑，不再卡顿假死
- ✅ 支持 **本地 NDK 文件**安装，离线环境也能使用
